# Biotech Landscape — Deep Analysis & Tracking Tool

## Overview
A comprehensive intelligence platform for mapping, analyzing, and tracking the global biotech landscape. Organized by sector and geography, with deep coverage of investment activity — who invests where, in what, when, and how much.

**Core purpose:** Give Bouke de Jong (PhD biotechnologist, entrepreneur) expert-level situational awareness of the biotech ecosystem to support strategic decisions, investment insights, partnership identification, and career positioning.

**Design principle:** Depth over breadth. Expert-level analysis, not a generic news aggregator.

## Data Entities

### 1. Companies
Each company record includes:

```
- Name
- Founded year
- HQ location + region
- Sector(s) [multi-tag]
- Technology platform
- Stage: Stealth / Early / Seed / Series A / B / C / D+ / Public / Acquired
- Employees (approx.)
- Total funding raised (USD/EUR)
- Latest valuation (if known)
- Key products / pipeline
- Business model: B2B / B2C / licensing / platform
- Partnerships & collaborations
- Key people (CEO, CSO, CTO)
- Website
- Notes / strategic observations
- Last updated

### 2. Investors
Each investor record includes:

```

- Type: VC / CVC / Family Office / Government / Accelerator / Angel
- HQ location
- Focus sectors [multi-tag]
- Focus stages: Pre-seed / Seed / Series A / B / Growth
- Geographic focus
- Fund size (if known)

- Typical check size (range)
- Co-investment preferences
- Website
- Notes
- Last updated

### 3. Investment Deals / Rounds
Each deal record includes:

```
- Company name
- Round type: Pre-seed / Seed / Series A / B / C / D+ / Grant / IPO / M&A

- Date (month/year)
- Lead investor(s)
- Co-investors
- Valuation (pre/post, if known)
- Use of funds (stated)
- Source / press link
- Notes / strategic significance
```

### 4. Trends & Themes
Tracked observations on sector movements:

```
- Theme name
- Related sectors
- Geographic momentum
- Key companies driving trend
- Key investors backing it
- Timeline / maturity stage
- Evidence / data points
- Bouke's analysis note
```

## Analysis Framework

### Landscape Mapping Dimensions
When analyzing a sector or company, always assess across:

1. **Technology readiness** — TRL 1–9, lab to commercial
2. **Market pull** — is there clear demand and a paying customer?
3. **Investment momentum** — number and size of recent deals
4. **Competitive density** — how crowded is the space?
5. **Geographic concentration** — where is activity clustering?
6. **Regulatory landscape** — what approvals/hurdles exist?
7. **Key bottlenecks** — what's holding the sector back?
8. **Strategic fit for Bouke** — relevance to her expertise and network

### Trend Signals to Monitor
- New funding rounds > €5M in target sectors
- Corporate acquisitions of biotech startups
- New EU/US regulatory approvals or frameworks
- Academic spin-outs from key institutions (Wageningen, Delft, MIT, ETH)
- Key hires at target companies (VP R&D, CSO appointments)
- New partnerships between corporates and startups
- Conference activity (Bioket, SynBioBeta, Bio-Europe, ESBES)
